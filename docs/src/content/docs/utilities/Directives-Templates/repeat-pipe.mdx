## Import

```ts
import { RepeatPipe } from 'ngxtension/repeat-pipe';
```

## Usage

### Basic

Use the `repeat` pipe as an easy way to loop over a number of iterations.

```ts
import { Component } from '@angular/core';
import { RepeatPipe } from 'ngxtension/repeat-pipe';

@Component({
	imports: [RepeatPipe],
	template: `
		<ul>
			@for (i of 3 | repeat; track i) {
				<li>{{ i }}</li>
			}
		</ul>
	`,
})
export class App {}
```

This will produce the following output:

```html
<!-- Output -->
<!-- <li>0</li> -->
<!-- <li>1</li> -->
<!-- <li>2</li> -->
```

### With `NgFor`

The pipe can also be utilized with the `ngFor` directive.

<Aside type="tip">
The  [`TrackByFunction`](https://angular.dev/api/core/TrackByFunction) **is not** automatically set.
Checkout the [Repeat Directive](/utilities/directives/repeat) as an alternative.

```ts
import { Component } from '@angular/core';
import { Repeat } from 'ngxtension/repeat';

@Component({
	imports: [Repeat],
	template: `
		<ul>
			<li *ngFor="let i; repeat: 3">{{ i }}</li>
		</ul>
	`,
})
export class App {}
```

This will produce the same output as the directive variant.

</Aside>

```ts
import { Component } from '@angular/core';
import { NgFor } from '@angular/common';
import { RepeatPipe } from 'ngxtension/repeat-pipe';

@Component({
	imports: [NgFor, RepeatPipe],
	template: `
		<ul>
			<li *ngFor="let i of 3 | repeat">{{ i }}</li>
		</ul>
	`,
})
export class App {}
```

This will produce the same output as above.

### Change start point

You can specify a `startAt` value as the second argument passed into the pipe.

```ts
import { Component } from '@angular/core';
import { RepeatPipe } from 'ngxtension/repeat-pipe';

@Component({
	imports: [RepeatPipe],
	template: `
		<ul>
			@for (i of 3 | repeat: 10; track i) {
				<li>{{ i }}</li>
			}
		</ul>
	`,
})
export class App {}
```

This will produce the following output:

```html
<!-- Output -->
<!-- <li>10</li> -->
<!-- <li>11</li> -->
<!-- <li>12</li> -->
```

### Descending sequence

Passing a **negative** `length` generates a descending sequence starting from `startAt`. The absolute value of `length` determines how many elements are produced.

```ts
import { Component } from '@angular/core';
import { RepeatPipe } from 'ngxtension/repeat-pipe';

@Component({
	imports: [RepeatPipe],
	template: `
		<ul>
			@for (year of -5 | repeat: 2026; track year) {
				<li>{{ year }}</li>
			}
		</ul>
	`,
})
export class App {}
```

This will produce the following output:

```html
<!-- Output -->
<!-- <li>2026</li> -->
<!-- <li>2025</li> -->
<!-- <li>2024</li> -->
<!-- <li>2023</li> -->
<!-- <li>2022</li> -->
```

## API

### Inputs

- `length: number` - An integer specifying the number of iterations. Positive values produce an ascending sequence, negative values produce a descending sequence, and zero produces an empty array.
- `startAt: number` - An integer specifying the start point of iterations. Defaults to `0`.

### Validation

- An error is thrown if `length` is not an integer.
- An error is thrown if `startAt` is not an integer.
